# gestion-de-servicios
Proyecto de graduacion


Pasos de la instalacion
1- Asegurarse de que se descargo completamente
2- Abrir la carpeta scripts y ejecutar el script 3-24-2020 DB_Servicios_Clean
3- Cambiar la cadena de conexion del API y conectarlo con la base de datos SITSA_FE (no conectar con produccion hasta que este testeado en dev)
4- Abrir el front end y ejecutar npm install para instalar todas las dependencias
5- Hacer el build del api y front end
6- Leer el manual de usuario para entender la funcionalidad
